# EcoStudy AI - Advanced Economics Tutor

EcoStudy AI is a modern web application designed to help students master complex Economics concepts, specifically focusing on Oligopoly Theory. It combines a sleek, premium user interface with a grounded AI agent powered by Gemini.

## üöÄ Features

- **Personalized AI Tutoring**: Grounded AI agent that provides context-aware answers to economics questions.
- **Resource Hub**: Curated list of PDFs and videos on Oligopoly, Game Theory, and Market Structures.
- **AI Analysis & Tips**: Quick insights and exam tips generated by AI.
- **Modern Dashboard**: High-fidelity dashboard built with React, Tailwind CSS, and Framer Motion.
- **Secure Authentication**: JWT-based authentication with bcrypt password hashing.

## üì¶ Setup & Installation

### Prerequisites
- Node.js (v18+)
- MongoDB Atlas account
- Gemini API Key

### 1. Clone the Repository
```bash
git clone <repository-url>
cd assignment1
```

### 2. Backend Setup
1. Navigate to the `server` directory:
   ```bash
   cd server
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the `server` directory:
   ```env
   PORT=5000
   MONGODB_URI=your_mongodb_connection_string
   GEMINI_API_KEY=your_gemini_api_key
   JWT_SECRET=your_jwt_secret
   ```
4. Start the server:
   ```bash
   npm run dev
   ```

### 3. Frontend Setup
1. Navigate to the `client` directory:
   ```bash
   cd ../client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

- **MongoDB Whitelisting**: Ensure your server IP is whitelisted in your MongoDB Atlas cluster.
